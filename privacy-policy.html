Privacy Policy

App: Dave's Persona Is Alive
Last updated: March 15, 2026

Overview

This is a personal automation tool that posts AI-generated content to social media platforms on behalf of a single authorized user. It is not a public-facing application and does not collect, store, or share data from any third party.

Data We Access

The application accesses LinkedIn profile data solely to post content on behalf of the authenticated user. No personal data is stored beyond what is required for authentication (access tokens stored locally on the user's own machine).

Data We Share

We do not share any data with third parties. The only external services used are:

  • LinkedIn API — to publish posts on behalf of the authorized user
  • Anthropic API — to generate post content (no personal data is transmitted)

Data Retention

No user data is retained by this application. All tokens are stored locally on the user's own server and are never transmitted to any external service other than the platform APIs they authenticate against.

User Control

The authorized user can revoke access at any time by deleting the LinkedIn access token from their LinkedIn Developer settings.

Contact

For any questions, contact the app owner directly via LinkedIn.
